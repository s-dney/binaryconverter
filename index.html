What is binary?

Binary is a way of representing numbers and other information using only two states (1 and 0). Binary is base 2. That means that instead of having a ones place (10⁰), a tens place (10¹), a hundreds place (10²) and so on, it has a ones place (2⁰), a twos place (2¹), a fours place (2²), and so on.
